# Images Directory

This directory contains image assets for the Snooker Management System.

## Structure

- `logo/` - Company and application logos
- `tables/` - Table images and layouts
- `members/` - Member profile pictures and related images
- `icons/` - Custom icons and graphics
- `backgrounds/` - Background images and patterns

## Guidelines

- Use optimized formats (WebP, PNG, JPG)
- Keep file sizes reasonable for web performance
- Use descriptive file names
- Organize by feature/component when possible

## Recommended Sizes

- Logo: 200x60px (main), 400x120px (high-res)
- Table images: 400x200px
- Member avatars: 150x150px
- Icons: 24x24px, 48x48px
- Backgrounds: 1920x1080px (or responsive)
