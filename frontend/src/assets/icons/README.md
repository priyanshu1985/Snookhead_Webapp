# Icons Directory

This directory contains custom icons and SVG assets for the Snooker Management System.

## Structure

- `navigation/` - Navigation and menu icons
- `actions/` - Action buttons and interactive icons
- `status/` - Status indicators and badges
- `equipment/` - Snooker equipment and table icons
- `general/` - General purpose icons

## Guidelines

- Use SVG format for scalability
- Follow consistent design language
- Use semantic naming conventions
- Include both outline and filled variants when appropriate
- Optimize SVGs for web performance

## Icon Sets

- Navigation: home, tables, bookings, members, billing, reports
- Actions: add, edit, delete, save, cancel, print, export
- Status: available, occupied, maintenance, confirmed, pending
- Equipment: cue, ball, table, scoreboard
- General: user, calendar, clock, money, settings
